以下の機能を持つ、C および C++ 向けのライブラリーです。

  • Json - RFC 8259に基づく pull 型の JSON パーザーのライブラリー。
  • Lattice - ビタビアルゴリズムと A*（エースター） アルゴリズムの実装です。
  • Text - 文字列処理ライブラリー。 現在は Unicode 13.0.0 に準拠した書記素分割器を提供します。
  • Trie - 共通接頭辞検索を行うことができるデータ構造で、ダブル配列で 実装しています。
