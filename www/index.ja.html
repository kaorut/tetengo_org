C および C++ 向けライブラリー

Version 1.2.0

  • Json

    • RFC 8259に基づく pull 型の JSON パーザーの ライブラリー。

  • Lattice

    • ビタビアルゴリズムとA*（エースター）アルゴリズムの実装です。

  • Text

    • 文字列処理ライブラリー。 現在は Unicode 13.0.0 に準拠した書記素分割器を 提供します。

  • Trie

    • 共通接頭辞検索を行うことができるデータ構造で、ダブル配列で実装しています。
