This library set for C and C++ provides the following features:

  • Json - A pull-style JSON parser library based on RFC 8259.
  • Lattice - An implementation of the Viterbi and the A* (A star) algorithm.
  • Text - A text processing library. This library now provides a grapheme splitter conforming to Unicode 13.0.0.
  • Trie - An associative data structure for common prefix search. Implemented with double array.
